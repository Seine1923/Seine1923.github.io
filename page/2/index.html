Java随机数

发表于 2015-10-23   |   分类于 Java   |  
java.lang.Math.Random和java.util.Random的随机函数都可以生成指定范围的不重复随机数，但从效率来讲，由于java.lang.Math.Random还需要浮点数乘法，以及强制类型转换，因此，显然java.util.Random效率更高。

而在java.util.Random中，一旦指定了同样的种子，那么无论什么时候运行，无论生成多长的随机数，都完全一致。通常按照默认指定种子为当前时间可避免这个问题，但是两个生成器之间尽量要有一定时间间隔，否则两者时间会一致，导致依然一样。
